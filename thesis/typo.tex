To make the text more lucid, following typographics conventions are used:
\begin{itemize}
\item \concept{Bold} face is used to type important informations. It also indicates new concepts which will be described further in the text
\item \app{Typewriter} face is used for program names, their input and output, files, directories and URLs
\item References are typed in the following: (ref)$_{(page)}$
\end{itemize}
