\textbf{Title}: Compression of XML Data\\
\textbf{Author}: \VT{}\\
\textbf{Department}: \KSI{}\\
\textbf{Supervisor}: \Pokorny{}\\
\textbf{Supervisor's e-mail address}: \email{pokorny@ksi.ms.mff.cuni.cz}\\
\textbf{Abstract}: XML data can be divided into elements. The probabilistic model of elements with the same type can be efficiently utilized during the modeling of the text and its subsequent compression. The aim of this work is to study the possibilities of XML data compression on the level of the element types, and to implement a system that makes experiments with various compression methods possible. A comparison with some of the existing XML-conscious compression systems is also provided.\\
\textbf{Keywords}: XML, data compression, grammar-based codes, string matching
